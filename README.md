# OperationsOn2numbers
-This project shows how to do simple math on two number.
-Both OOP and simple

##When run
-The program will ask for 2 numbers from the user
-It will then do the following and output each result:

###Add 
-num1 + num2

###Subtract
-num1 - num2

###Multiply
-num1 * num2

###Divide
-num1 / num2
